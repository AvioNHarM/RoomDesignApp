## Getting Started

### Backend

1. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```
2. **Run migrations:**
   ```bash
   python manage.py migrate
   ```
3. **Start the server:**
   ```bash
   python manage.py runserver
   ```

### Frontend

1. **Install dependencies:**
   ```bash
   npm install
   ```
2. **Start the development server:**
   ```bash
   npm run dev
   ```
3. **Open [http://localhost:3000](http://localhost:3000) in your browser.**
